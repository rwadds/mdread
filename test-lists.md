# List tests

Exercises nested lists, ordered and unordered mixing, task checkboxes, custom
start numbers, and continuation lines.

## Deep nesting

- Level one
  - Level two
    - Level three
      - Level four
- Back to level one

## Ordered within unordered within ordered

1. First step
   - a bullet under step one
     1. a number under the bullet
     2. a second number
   - another bullet
2. Second step

## Each bullet marker

- dash marker
* star marker
+ plus marker

## Ordered list starting at five

5. five
6. six
7. seven

## Ordered list with the `)` delimiter

1) one
2) two
3) three

## Task list

- [ ] An open task
- [x] A task done, lowercase x
- [X] A task done, uppercase X
- [ ] Another open task

## Tasks nested under tasks

- [ ] Parent task
  - [x] Subtask, finished
  - [ ] Subtask, still open
    - [x] A sub-subtask
- [x] A second parent, done

## Loose list — blank lines between items

- First item

- Second item

- Third item

## Continuation lines

- This item's text runs onto
  a second indented line and should be joined into one item.
- A short item that stands alone.
